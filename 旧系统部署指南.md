# AC-Nexus 旧系统24小时部署指南

适用于 Windows 7 / 8 / 8.1、旧版 Linux、树莓派等无法运行 PySide6 GUI 的系统。

## 原理

AC-Nexus 核心库（`acnexus_core/`）**零 GUI 依赖**，只需要 Python + `broadlink` 就能完成全部自动控制功能。GUI 只是操作界面，不影响后台运行。

```
acnexus_core/          ← 核心，旧系统能跑
  ├── ac_control.py    ← 空调控制
  ├── scheduler.py     ← 定时任务
  ├── typhoon.py       ← 台风关机
  ├── xiaomi_local.py  ← 米家控制
  └── config.py        ← 配置文件

acnexus_desktop/       ← GUI，旧系统不需要
```

## 方案 A：Windows 7（推荐）

### 1. 安装 Python 3.8

> Python 3.8 是最后支持 Windows 7 的版本。

下载：https://www.python.org/ftp/python/3.8.10/python-3.8.10-amd64.exe

安装时勾选 **Add Python to PATH**。

### 2. 准备配置文件

先在正常电脑上运行 AC-Nexus 桌面版，完成首次设置（扫描设备、配置品牌、设置定时模板）。关闭程序后，将配置文件复制到老电脑：

```
从正常电脑复制:
  %USERPROFILE%\.ac_controller\config.json
  %USERPROFILE%\.ac_controller\logs\        （可选）

粘贴到老电脑:
  C:\Users\<用户名>\.ac_controller\config.json
  C:\Users\<用户名>\.ac_controller\logs\
```

### 3. 安装依赖

```cmd
pip install broadlink pycryptodome
```

如需米家 MIoT 支持：

```cmd
pip install pycryptodome
```

### 4. 创建启动脚本

在老电脑任意位置创建 `acnexus_service.py`：

```python
"""
AC-Nexus 后台守护进程
24小时运行，无需 GUI
"""
import sys, os, time

# 将源代码目录加入路径（改成你实际存放的路径）
SRC_DIR = r"C:\Users\<用户名>\Desktop\AC-Nexus"
if SRC_DIR not in sys.path:
    sys.path.insert(0, SRC_DIR)

from acnexus_core import init
from acnexus_core.scheduler import register_all_jobs, start_scheduler
from acnexus_core.typhoon import fetch_and_cache, judge_and_shutdown
from acnexus_core.logger import write_log

# 初始化
init()
register_all_jobs()
start_scheduler()

print("AC-Nexus 后台模式启动成功")
write_log("系统", "AC-Nexus 后台模式启动")

# 首次拉取台风数据
try:
    fetch_and_cache()
    judge_and_shutdown(write_log)
except Exception:
    pass

# 主循环
try:
    while True:
        # 每 30 分钟检查一次风暴
        try:
            fetch_and_cache()
            judge_and_shutdown(write_log)
        except Exception as e:
            write_log("系统", f"风暴检查异常: {e}")
        time.sleep(1800)
except KeyboardInterrupt:
    write_log("系统", "AC-Nexus 手动停止")
    print("已退出")
```

### 5. 设置开机自启（Windows 7）

创建 `acnexus_start.bat`：

```bat
@echo off
start /B pythonw C:\Users\<用户名>\acnexus_service.py > nul 2>&1
```

将此 `bat` 文件放到**启动文件夹**：
- 按 `Win+R` → 输入 `shell:startup` → 回车
- 拖入 `acnexus_start.bat`

### 6. 验证运行

查看日志文件确认正在运行：

```
C:\Users\<用户名>\.ac_controller\logs\2026-06-26.md
```

- 看到 `AC-Nexus 后台模式启动` 说明成功
- 看到 `⏰ [设备名] 定时触发` 说明定时任务正常
- 看到 `[HH:MM] 台风自动关机完成` 说明台风监控正常

## 方案 B：树莓派 / 旧 Linux（推荐）

### 1. 安装依赖

```bash
sudo apt update
sudo apt install python3 python3-pip -y
pip3 install broadlink pycryptodome
```

### 2. 复制配置文件

```bash
mkdir -p ~/.ac_controller/logs
# 用 U 盘或 scp 复制 config.json 到 ~/.ac_controller/
```

### 3. 创建 systemd 服务

```bash
sudo nano /etc/systemd/system/acnexus.service
```

写入：

```ini
[Unit]
Description=AC-Nexus Background Service
After=network.target

[Service]
Type=simple
ExecStart=/usr/bin/python3 /home/pi/acnexus_service.py
WorkingDirectory=/home/pi
User=pi
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动：

```bash
sudo systemctl daemon-reload
sudo systemctl enable acnexus
sudo systemctl start acnexus
sudo systemctl status acnexus   # 查看状态
```

## 方案 C：OpenWRT 路由器

### 1. 安装 Python

```bash
opkg update
opkg install python3 python3-pip
pip3 install broadlink pycryptodome
```

### 2. 部署脚本

```bash
# SSH 到路由器
scp acnexus_service.py root@192.168.1.1:/
scp -r acnexus_core/ root@192.168.1.1:/
ssh root@192.168.1.1 "python3 /acnexus_service.py &"
```

## 限制与说明

| 功能 | 后台模式 | 说明 |
|------|:--:|------|
| 定时开关机 | ✅ | 调度器全功能运行 |
| 台风监控 | ✅ | 每 30 分钟检查 |
| 风暴自动关机 | ✅ | 到达阈值自动触发 |
| 自动调温 | ✅ | 根据天气自动调整 |
| 手动发码 | ❌ | 需要 GUI 或 Agent 调用 |
| 查看/修改设备 | ❌ | 编辑 `config.json` 或 SSH |
| 查看日志 | ✅ | 直接看 `logs/*.md` |
| 编辑定时模板 | ❌ | 编辑 `config.json` 中的 `schedule_templates` |

## 日常管理

| 需求 | 方法 |
|------|------|
| 查看运行状态 | 看 `logs/` 下今天的 `.md` 文件 |
| 修改定时 | 编辑 `config.json` 的 `schedule_templates`，保存后自动生效 |
| 临时手动操作 | 写一个简单脚本调 `send_ac()` |
| 更新程序 | 替换源代码文件，重启后台进程 |
