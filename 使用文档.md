# BroadlinkAC 使用文档

BroadlinkAC 是一个用博联（Broadlink）红外遥控器控制空调的电脑软件。可以手动开关空调、根据室外温度自动调温、设置定时开关、查看台风和天气预报。

支持品牌：格力、美的、海尔、奥克斯、海信、大金、三菱、松下、华凌、小米

---

## 怎么运行

### Windows

下载 `BroadlinkAC.exe`，双击打开就行，不用装 Python。

### macOS

下载 `BroadlinkAC.app.zip`，解压后拖到 `/Applications` 文件夹。

首次打开如果提示"无法验证开发者"，在终端执行：

```bash
xattr -cr /Applications/BroadlinkAC.app
```

然后重新双击打开即可。

### Linux

下载源码后：

```bash
# 先装依赖
pip install -r requirements.txt

# 启动
python ac_controller_pyside6.py
```

### 从源码运行（全平台通用）

```bash
# 安装依赖
pip install -r requirements.txt

# 启动程序
python ac_controller_pyside6.py
```

## 第一次使用

打开软件后，按顺序做这几件事：

### 1. 扫描设备

点顶部工具栏的 **扫描**，软件会自动找局域网里的博联设备。找到后会自动添加。

如果没找到，检查：
- 电脑和博联是不是连的同一个 WiFi
- 防火墙有没有挡掉

### 2. 设置天气

点右上角 **⚙ 设置**：

- 选一个天气来源（百度天气或和风天气）
- 填上对应的 API Key
- 点 **自动定位** 获取你所在的城市

两个来源都不用花钱，注册一下就能拿到 Key：

| 来源 | 注册地址 |
|------|---------|
| 百度天气 | [百度地图控制台](https://lbsyun.baidu.com/apiconsole/key) |
| 和风天气 | [和风天气控制台](https://console.qweather.com) |

### 3. 选空调品牌

在设置里选对空调品牌（默认格力）。不同品牌用的红外码不一样，选错了可能控制不了。

## 主界面介绍

软件有四个标签页，通过底部按钮切换：

### 🎮 空调控制

**上半部分是天气卡片**：显示当前室外温度、体感、湿度、风向。右边有当前天气的图标（晴、多云、雨等），点刷新可以更新数据。

**下半部分是控制面板**：

- **开关**：点左上角的电源按钮开关空调
- **模式**：制冷、制热、除湿、送风、自动
- **温度**：点 + - 按钮调温度，范围 16~30°C
- **风速**：自动、低、中、高

调好之后不需要再点"发送"，切换模式或温度会自动发送指令。状态栏左下角会显示结果。

**右边是智能规则**：软件会根据室外温度自动选模式和温度。比如室外 36°C 时自动开 24°C 制冷，室外 20°C 时自动关空调。点 **编辑规则** 可以自己改。

**底部是定时任务**：下拉框可以快速切换定时模板，下方显示当前模板的详情。点 **编辑定时** 可以创建和修改模板。

### ⚠️ 预警信息

左边是台风监测，右边是天气预警，左右各占一半。

台风卡片显示：
- 台风名称、编号、等级、气压、风速
- 移动方向和速度
- 离你多远，是正在靠近还是远离
- 路径预报图（如果有预报数据的话）

底部可以设置预警距离和自动关机：
- 台风进入设定距离时弹窗提醒
- 风暴小于 100km 时自动关空调（保护外机）

数据来自中央气象台和天气 API。

记录软件的运行日志，按时间排序，可以搜索。

## 定时开关机

定时功能用"模板"管理，一个模板可以包含多组时间。比如：

- **工作日**：周一到周五，早上 8 点开，下午 6 点关
- **周末**：周六周日，中午 12 点开，晚上 11 点关

### 创建模板

1. 在空调控制页面的定时区域，点 **编辑定时**
2. 点 **＋** 新建模板，取个名字
3. 勾选星期几生效
4. 设置开机和关机的时间
5. 如果需要再加一组日期（比如周末不同时间），点 **+ 添加日期组**
6. 点 **保存**

每个时段可以单独开关（比如只设开机不设关机）。

### 切换模板

在定时区域的下拉框直接选就行，会自动保存并生效。选 **未启用** 可以关掉定时。

### 自动调温

定时任务下面的滑动开关，打开后每 2 小时自动根据室外温度调整空调。这个功能需要天气 API 正常工作。

## 故障排查

### 搜不到设备

1. 确认在同一个 WiFi
2. 防火墙放开 UDP
3. 重启博联设备

### 天气显示不出来

1. 检查 API Key 填对了没
2. 在设置里切换到另一个天气来源试试

### 定时不生效

1. 确认模板已选上（不是"未启用"）
2. 确认设备在线
3. 检查时段的时间对不对

### 空调按了没反应

1. 确认品牌选对了
2. 确认博联设备的红外发射灯有闪
3. 试试换个位置摆放博联
