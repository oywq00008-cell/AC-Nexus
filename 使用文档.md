# BroadlinkAC v3A 使用文档

BroadlinkAC 是一款基于博联（Broadlink）红外遥控设备的智能空调控制器，支持多品牌多设备控制、室外天气监测、台风预警和定时开关机。

---

## 支持的空调品牌

格力、美的、海尔、奥克斯、海信、大金、三菱、松下、华凌、小米

---

## 安装与启动

### 方式一：下载安装包（推荐）

| 平台 | 安装方式 |
|------|----------|
| **Windows** | 下载 `BroadlinkAC.exe`，双击启动，无需安装 Python |
| **macOS** | 下载 `BroadlinkAC.app.zip`，解压拖入 `/Applications` |

#### macOS 验证问题

首次打开可能提示「无法验证开发者」，在终端执行：

```bash
xattr -cr /Applications/BroadlinkAC.app
```

然后重新双击打开即可。

### 方式二：源码运行（Windows / macOS / Linux）

```bash
pip install -r requirements.txt
python ac_controller.py
```

## 配置步骤

点击顶部菜单栏 **⚙️ 设置**：

### 1. 天气 API（支持双源）

软件通过天气 API 获取室外温度和当地天气预警，支持两种数据源：

| 数据源 | 免费额度 | 注册地址 |
|--------|---------|---------|
| 百度天气 | 5,000 次/天 | [百度地图控制台](https://lbsyun.baidu.com/apiconsole/key) |
| 和风天气 | 50,000 次/月 | [和风天气控制台](https://console.qweather.com) |

- 填入对应 API Key
- 和风免费订阅需填写个人 Host 地址（格式 `https://xxx.re.qweatherapi.com`）
- 可在设置中切换数据源或勾选"百度"/"和风"指定默认
- 未指定时自动降级：百度不可用 → 回退和风

### 2. 城市定位

点击 **自动定位** 按钮获取当前位置，或手动输入城市名搜索。

### 3. 空调品牌

在下拉菜单中选择空调品牌。品牌决定红外码协议，**每个设备可独立设置不同品牌**。

## 多设备管理

### 设备下拉栏

软件顶部栏左侧显示所有已添加的设备，点击切换：

```
📡 设备: [RM4 mini ▼] [✏️] [✕] [🔍 扫描]
```

### 扫描发现

点击 **🔍 扫描** 自动发现局域网中的博联设备，新设备会自动添加。启动后也会自动扫描一次。

### 设备改名

点击 **✏️** → 输入新昵称 → 保存。昵称仅用于显示，不影响设备识别，重新扫描不会覆盖。

### 删除设备

点击 **✕** → 确认删除。至少保留一台设备。

### 离线检测

扫描后，未在线的设备会标注 **(离线)**。定时任务执行到离线设备时自动跳过，等待下个周期重试。

### 独立配置

每台设备拥有独立的品牌、定时、规则设置，切换设备不会影响其他设备的运行中的调度任务。

## 空调控制

### 手动模式

在 **🎮 空调控制** 标签页：

- 选择 **模式**：制冷 / 制热 / 除湿 / 送风 / 自动
- 设定 **温度**：16°C ~ 30°C
- 控制 **风速**：自动 / 1 档 / 2 档 / 3 档
- 点击 **发送** 向**当前选中设备**发射红外指令

控制面板右侧显示当前品牌 Logo 图标。

---

## 预警信息

点击 **⚠️ 预警信息** 标签页：

### 台风监测

- 显示西北太平洋当前活跃台风
- **路径预报图**：Canvas 矢量路径图，可视化显示台风当前→未来48h移动轨迹
  - 🌀 emoji 标注当前位置
  - 灰色虚线连接预报路径
  - 橙色虚线直连你家，中标注距离
  - 距离行动态显示 🔴 正在靠近 / 🟢 正在远离 / ⚪ 徘徊
- 可设置台风预警距离（默认 800km）
- 点击 **🌍 卫星云图** 在浏览器查看云图
- 台风进入预警范围时弹窗提醒，支持"本次启动不再显示"

> 数据来源：中央气象台

### 当地天气预警

- 显示当前位置的天气预警（高温/暴雨/大风等）
- 预警卡片按严重等级（🔴红/🟠橙/🟡黄/🔵蓝）排序
- 数据来源标识自动匹配实际使用的 API

> 数据来源：百度天气 / 和风天气（根据设置）

## 定时开关机

在空调控制标签页的 **定时计划** 区域：

- **定时开机**：设置每天自动开机的时间
- **定时关机**：设置每天自动关机的时间
- 分别有启用/关闭开关
- **每台设备独立配置**

---

## 开机自启

在设置中打开 **开机自启动**：

- **Windows**：写入注册表 Run 键
- **macOS**：创建 LaunchAgent

---

## 系统托盘

- 点击窗口 **关闭按钮（X）** → 隐藏到任务栏右下角托盘
- 双击托盘图标 → 恢复窗口
- 右键托盘图标 → **显示** / **退出**

## 外观主题

在设置中可切换 **浅色 / 深色 / 跟随系统** 三种主题。

## 故障诊断

点击菜单栏 **🔧 诊断** 自动检测：

- Python 环境版本
- 依赖库完整性（缺库可一键安装）
- 博联设备扫描与认证
- 网络连通性（本机→网关→外网三层递进）
- 天气 API 状态

## 常见问题排查

### 搜索不到博联设备

1. 确认电脑和博联设备连接到 **同一局域网**
2. 检查防火墙是否允许 UDP 通信
3. 使用 **🔧 诊断** 工具自动扫描

### 台风数据加载失败

1. 检查网络连接
2. 中央气象台偶尔服务器维护会短暂不可用

### 天气 API 无响应

1. 确认已填写 API Key
2. 使用 **🔧 诊断** 检测 API 状态
3. 可尝试切换数据源（百度 ↔ 和风）

### 定时开关机不生效

1. 确认定时开关已打开
2. 确认设备已在线（无"(离线)"标记）
3. 检查时间格式是否正确

### 文字看不清

在设置中切换外观主题（浅色/深色），或选择"跟随系统"自动适配。
