# BroadlinkAC 使用文档

BroadlinkAC 是一个用博联（Broadlink）红外遥控器控制空调的电脑软件。可以手动开关空调、根据室外温度自动调温、设置定时开关、查看台风和天气预报。

支持品牌：格力、美的、海尔、奥克斯、海信、大金、三菱、松下、华凌、小米

---

## 怎么运行

### Windows

下载 `BroadlinkAC.exe`，双击打开就行，不用装 Python。

### macOS

下载 `BroadlinkAC.app.zip`，解压后拖到 `/Applications` 文件夹即可。

### Linux

下载源码后，打开终端：

```bash
# 先装依赖（只需要做一次）
pip install -r requirements.txt

# 启动
python ac_controller_pyside6.py
```

## 第一次使用

打开软件后，按顺序做这几件事：

### 1. 扫描设备

点顶部工具栏的 **扫描**，软件会自动在局域网里找博联设备。找到后会自动添加。

> 如果没找到，点菜单栏的 **诊断 → 故障诊断**，软件会帮你排查原因。

### 2. 设置天气

点菜单栏 **设置 → 设置**：

- 选一个天气来源（百度天气或和风天气）
- 填上对应的 API Key（下面有免费获取方法）
- 点 **自动定位** 获取你所在的城市

两个来源都免费，注册一下就能拿到 Key：

| 来源 | 怎么拿到 Key |
|------|-------------|
| 百度天气 | 打开 [百度地图控制台](https://lbsyun.baidu.com/apiconsole/key)，注册后创建应用，选"服务端"，复制 AK |
| 和风天气 | 打开 [和风天气控制台](https://console.qweather.com)，注册后创建项目，复制 API Key |

> 看不懂注册页面？没关系，就是填个名字、选个类型、点创建，然后把那一长串字母数字复制过来就行。

### 3. 选空调品牌

在设置里选对空调品牌（默认格力）。不同品牌用的红外码不一样，选错了可能控制不了。

> 如果列表里没有你的品牌，下面有 **自定义遥控器** 的方法。

### 4. 自定义遥控器

如果你的空调品牌不在内置列表中，可以通过红外学习功能，用空调原装遥控器"教"软件发码。

#### 新建自定义遥控器

1. 打开 **设置**，在基础设置卡片中找到 **自定义遥控器** 那一行
2. 点最右边的 **＋ 按钮**（新增自定义遥控器）
3. 输入设备名称（比如"客厅空调"）
4. 可选选一个品牌 Logo（与哪个内置品牌长相相似就选哪个，没 Logo 也行）
5. 用 **模式 + 温度 + 风速** 三个下拉框组合出要学习的指令，点 **＋ 添加**
   - 比如选"制冷 + 26 + 自动"，就是学习"制冷 26°C 自动风"这条指令
   - 关机指令会自动添加，不用手动加
6. 添加完所有需要的组合后，点 **开始学习**

#### 学习步骤

点开始学习后会弹出向导窗口，按顺序一步步操作：

1. **步骤 1 — 关机**：先打开空调遥控器，然后对准红外设备按遥控器的 **关机键**
2. 点 **开始捕获** 按钮，遥控器按键时软件会自动接收红外信号
3. 每完成一步，窗口自动跳到下一步（制冷_26_自动、制冷_24_自动……）
4. 最后一步完成后，所有学到的指令会保存到本地

> 提示：学习时把遥控器对准博联设备的红外接收窗，距离不要太远。如果某一步没学到，可以关掉重新来。

#### 编辑和删除

- 选中自定义遥控器后，右侧会出现 **编辑按钮（蓝色）** 和 **删除按钮（红色）**
- 编辑可以增减要学习的组合、修改名称
- 删除会连同学到的所有码一起清除，不可恢复

> 自定义遥控器的码保存在 `~/.ac_controller/custom_codes.json`，重装系统前建议备份这个文件。

#### 温度规则检测

使用自定义遥控器时，软件会自动检查你学过的指令是否够用：

- 在 **编辑温度规则** 点保存时，软件会逐条检查每条规则需要的指令（比如"制冷 24°C 自动风"、"制热 28°C 自动风"）
- 如果发现某个组合还没学过，会弹出警告窗口，列出所有缺失的指令
- 你可以选择 **取消** 先回去学习，也可以 **仍然保存**（定时和自动调温遇到缺失指令时会跳过）

> 温度规则同时服务于两个功能：**自动调温**（根据实时室外温度切换模式）和 **定时开机**（到点后根据当前外温匹配规则决定制冷/制热）。如果某条规则对应的指令没学过，到了该温区定时开机可能不会执行。建议至少学习：制冷 24~28°C + 制热 26~30°C + 自动风。温度跨度越大，自动运行越不受限制。

## 主界面介绍

软件有两个标签页，通过底部按钮切换：

### 🎮 空调控制

**上半部分是天气卡片**：显示当前室外温度、体感、湿度、风向。点刷新可以更新数据。

**下半部分是控制面板**：

- **开关**：点电源按钮开关空调
- **模式**：制冷、制热、除湿、送风、自动
- **温度**：点 + - 按钮调温度，范围 16~30°C
- **风速**：自动、低、中、高

状态栏左下角会显示发送结果。

**右边是智能规则**：软件会根据室外温度自动选模式和温度。比如室外 36°C 时自动开 24°C 制冷，室外 20°C 时自动关空调。点 **编辑规则** 可以自己改。

**底部是定时任务**：下拉框可以快速切换定时模板，下方显示当前模板的详情。选 **关闭定时** 可以关掉所有定时。点 **编辑定时** 可以创建和修改模板。

### ⚠️ 预警信息

左边是风暴监测，右边是天气预警。

台风卡片显示台风名称、等级、距离、移动方向，还有路径预报图。底部可以设置：

- 台风进入设定距离时弹窗提醒
- 风暴小于 100km 时自动关空调（保护外机）

## 定时开关机

定时功能用"模板"管理，一个模板可以包含多组时间。比如：

- **工作日**：周一到周五，早上 8 点开，下午 6 点关
- **周末**：周六周日，中午 12 点开，晚上 11 点关

### 创建模板

1. 在空调控制页面的定时区域，点 **编辑定时**
2. 点 **＋** 新建模板，取个名字
3. 勾选星期几生效
4. 设置开机和关机的时间
5. 如果需要再加一组日期（比如周末不同时间），点 **+ 添加日期组**
6. 点 **保存**

每个时段可以单独开关（比如只设开机不设关机）。

### 切换模板

在定时区域的下拉框直接选就行，会自动保存并生效。选 **关闭定时** 可以完全关掉定时。

## 遇到问题怎么办

### 🩺 最快的方法：故障诊断

点菜单栏 **诊断 → 故障诊断**，点 **开始诊断**。软件会自动检查：

- Python 环境是否正常
- 依赖包是否装全
- 能不能找到博联设备
- 网络通不通
- 天气 API Key 能不能用

每一项后面有 ✅ 或 ❌。看到 ❌ 就对照下面的说明解决。

---

### 搜不到设备

1. 确认电脑和博联连的是同一个 WiFi（不是同一个就不行）
2. 重启一下博联设备（拔掉电源再插上）
3. 如果还不行，检查防病毒软件或防火墙有没有拦截
4. 实在不行，打开终端输入 `ping 192.168.8.214`（把 IP 换成你博联的地址），看通不通

### 天气显示不出来

1. 打开设置，检查 API Key 填对了没有（不能多空格、不能漏字母）
2. 确认 Key 在官网是"启用"状态（刚注册的可能需要等几分钟生效）
3. 可以切换到另一个天气来源试一下
4. 检查网络通不通（用诊断功能）

### 显示"无法连接外网"

1. 确认电脑连着 WiFi 或插着网线
2. 打开浏览器，随便访问一个网页（比如 www.baidu.com），看能不能打开
3. 如果网页能打开但诊断还是报错，可能是防病毒软件拦了软件的网络请求，把 BroadlinkAC 加入白名单
4. 公司/学校网络有时会限制，尝试切换手机热点测试

### 提示"依赖未安装"

这是打包或安装不完整。解决方法：

- **Windows/macOS 用户**：重新下载安装包，覆盖安装
- **Linux 用户**：在终端运行 `pip install -r requirements.txt`

### 定时不生效

1. 确认模板已选上（下拉框显示的不能是"关闭定时"）
2. 确认设备在线
3. 检查时段的时间对不对（别把上午下午搞混了）
4. **使用自定义遥控器时**：定时开机需要根据室外温度匹配规则，如果规则对应的指令没学过，到点不会执行。建议在编辑规则点保存时留意弹窗提示，把缺失的组合学上

### 空调按了没反应

1. 确认品牌选对了（设置里可以看到当前品牌）
2. 确认博联设备的红外发射灯有闪（按下发送时灯应该亮一下）
3. 试试换个位置摆放博联（不要离空调太远、中间不要有遮挡）
4. 检查空调是不是处于红外接收状态（有的空调关机后不接收信号）
