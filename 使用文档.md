# BroadlinkAC v3 使用文档

BroadlinkAC 是一款基于博联（Broadlink）红外遥控设备的智能空调控制器，支持多品牌空调控制、室外天气监测、台风预警和定时开关机。

---

## 支持的空调品牌

格力、美的、海尔、奥克斯、海信、大金、三菱、松下、华凌、小米

---

## 安装与启动

### 方式一：下载安装包（推荐）

| 平台 | 安装方式 |
|------|----------|
| **Windows** | 下载 `BroadlinkAC.exe`，双击启动，无需安装 Python |
| **macOS** | 下载 `BroadlinkAC.app.zip`，解压拖入 `/Applications` |

#### macOS 验证问题

首次打开可能提示「无法验证开发者」，在终端执行：

```bash
xattr -cr /Applications/BroadlinkAC.app
```

然后重新双击打开即可。

### 方式二：源码运行（Windows / macOS / Linux）

```bash
pip install -r requirements.txt
python ac_controller.py
```

## 配置步骤

点击菜单栏 **⚙️ 设置**：

### 1. 和风天气 API

软件通过和风天气获取室外温度和当地天气预警，需要免费注册：

1. 前往 [和风天气控制台](https://console.qweather.com)
2. 注册并创建免费 API Key
3. 在设置中填入 **API Key**
4. 在和风控制台 → **设置 → API Host** 中复制 **个人 Host 地址**，填入设置

> 💡 免费订阅的用户必须填写个人 Host 地址，格式如 `https://xxx.re.qweatherapi.com`

### 2. 城市定位

点击 **自动定位** 按钮获取当前位置，或手动输入城市名搜索。

### 3. 空调品牌

在下拉菜单中选择你使用的空调品牌。品牌决定了红外码协议。

## 空调控制

### 手动模式

在 **🎮 空调控制** 标签页：

- 选择 **模式**：制冷 / 制热 / 除湿 / 送风 / 自动
- 设定 **温度**：16°C ~ 30°C
- 控制 **风速**：自动 / 1 档 / 2 档 / 3 档
- 点击 **发送** 发出红外指令

### 自动模式

软件根据室外温度自动决定开关和模式，规则可在设置中自定义。开启 **自动调节** 后，每天定时根据天气自动执行。

---

## 预警信息

点击 **⚠️ 预警信息** 标签页：

### 台风监测

- 显示西北太平洋当前活跃台风
- 点击台风名称查看详情、路径预报和离你有多远
- 可设置台风预警距离（默认 800km）
- 点击 **🌏 卫星云图** 在浏览器查看云图

> 数据来源：中央气象台

### 当地天气预警

- 显示当前位置的天气预警（高温/暴雨/大风等）
- 点击 **🔄 刷新预警** 获取最新数据

> 数据来源：和风天气

## 定时开关机

在空调控制标签页的 **定时计划** 区域：

- **定时开机**：设置每天自动开机的时间
- **定时关机**：设置每天自动关机的时间
- 分别有启用/关闭开关

## 开机自启

在设置中打开 **开机自启动**：

- **Windows**：写入注册表 Run 键
- **macOS**：创建 LaunchAgent

## 系统托盘（Windows）

- 点击窗口 **关闭按钮（X）** → 隐藏到任务栏右下角托盘
- 双击托盘图标 → 恢复窗口
- 右键托盘图标 → **显示** / **退出**

## 故障诊断

点击菜单栏 **🔧 诊断** 自动检测：

- Python 环境版本
- 依赖库完整性（缺库可一键安装）
- 博联设备扫描与认证
- 网络连通性（本机→网关→外网三层递进）
- 和风天气 API 状态

## 常见问题排查

### 搜索不到博联设备

1. 确认电脑和博联设备连接到 **同一局域网**（同一个路由器）
2. 检查 Windows 防火墙是否允许 Python 或 exe 的 UDP 通信
3. 使用 **🔧 诊断** 工具自动扫描
4. 如果 IP 变更，诊断工具会自动更新缓存

### 台风数据加载失败

1. 检查网络连接（使用诊断工具测试连通性）
2. 台风数据来自中央气象台，偶尔服务器维护会短暂不可用

### 天气 API 无响应

1. 确认已在设置中填写了 **和风 API Key** 和 **个人 Host**
   - Host 地址在和风控制台 → 设置 → API Host 中查看
2. 使用 **🔧 诊断** 工具检测 API 状态
3. 如果显示网络问题，诊断会自动 ping 网关和外网定位故障

### 文字看不清（颜色与背景同色）

在设置中切换外观主题（浅色/深色），或选择"跟随系统"自动适配。

### 定时开关机不生效

1. 确认定时开关旁边的开关已打开（绿色）
2. 检查时间格式是否正确

### 开机自启无效

- **Windows**：以管理员身份运行一次程序，或检查杀毒软件是否阻止
- **macOS**：检查系统偏好设置 → 用户与群组 → 登录项

### 软件无响应/卡死

诊断窗口、天气刷新等操作已做异步处理，通常在几秒内完成。如果长时间无响应，重启程序即可。

### macOS 无法打开「无法验证开发者」

这是 macOS 的安全机制。在终端执行以下命令即可：

```bash
xattr -cr /Applications/BroadlinkAC.app
```

然后重新双击打开。
