### 关键架构理解
- 入口：`ac_controller_pyside6.py` → `broadlinkac_desktop.app_pyside6:main()`
- Core API：`broadlinkac_core` — `init()`, `send_ac()`, `fetch_weather()`, `typhoon_threat_distance()`
- 品牌解析：`resolve_brand()` 支持中英文名 → hvac_ir 或 protocols 模块
- 配置：原子写入 `~/.ac_controller/config.json`，多设备支持，设备扁平键管理
- 调度：基于 `schedule` 库的后台守护线程，多日期组模板系统
- 天气：百度 + 和风 双源，自动降级
- 风暴：NMC（西北太平洋）+ NHC（北大西洋）双源
- GUI：PySide6 Fusion 风格，自定义 Tab 栏，浅色/深色主题
- 红外：ACProtocol 基类 + 14 个品牌子类（hvac_ir），3 个自研协议（protocols/）
- 打包：PyInstaller 三平台 spec，CI/CD GitHub Actions
